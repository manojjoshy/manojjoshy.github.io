Designing the Visual System for Markers Pro

Role: UX designer, UI designer
Date: Aug 2020 - Dec 2020
Team: Thulasiraman P, Anup Zade, Manoj Joshi, Abhishek Ghosh
Tools: Photoshop, Microsoft Teams, Jira, Sharepoint

Overview

MarkersPro, a comprehensive school management software, faced the challenge of creating a consistent and user-friendly visual system across its diverse modules. With functionalities spanning from administrative tasks to academic management, the software required a cohesive design that enhanced usability and efficiency. The primary challenge was to align the design with the existing Bootstrap framework while meeting the client’s requirements for adaptability and intuitive user interactions.

MarkersPro offers a wide range of modules designed to address various administrative and academic needs within educational institutions. These include

  • Admin
  • Assessments
  • Assignments
  • Attendance
  • Billing
  • Communication
  • Learning Management System
  • Discipline
  • District Module
  • Communication
  • Reporting
  • Scheduling
  • Staff
  • Students
  • Transportation
  • Cafeteria

Together, these modules create a comprehensive and integrated school management system.

Problem Statement & Hypothesis

Problem Statement

The challenge was to design a cohesive and user-friendly visual system for MarkersPro that would ensure consistency and clarity across its diverse range of modules, while aligning with the existing Bootstrap framework and client requirements.

Hypothesis

By creating a unified visual language—including a form UI kit, color kit, typography, iconography, and a base template—MarkersPro will achieve a consistent user experience, improving usability and efficiency for its end-users.

Design Objectives

Develop a visual system that ensures consistency across all modules.

Create an intuitive and accessible user interface that supports various functions of the software.

Integrate seamlessly with the Bootstrap framework for a responsive and adaptable design.

Design Process

Design Components

Form UI Kit:

  • Designed various form elements such as input fields, buttons, checkboxes, and radio buttons.
  • Ensured clear labeling, proper spacing, and consistent interaction states (hover, focus, disabled).

Bootstrap UI Kit:

  • Customized Bootstrap components to align with the MarkersPro visual style.
  • Developed reusable components like cards, modals, and buttons, maintaining consistency throughout the application.
